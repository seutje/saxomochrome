Buffer → Tone

Buffer loading and storage. Tone.Buffer is used internally by all classes that make requests for audio files such as Tone.Player, Tone.Sampler and Tone.Convolver.

Aside from load callbacks from individual buffers, Tone.Buffer provides static methods which keep track of the loading progress of all of the buffers. These methods are Tone.Buffer.onload, Tone.Buffer.onprogress, and Tone.Buffer.onerror.

new Tone.Buffer (url[, onload])
url {AudioBuffer|string}

The url to load, or the audio buffer to set.

onload {function}

A callback which is invoked after the buffer is loaded. It's recommended to use Tone.Buffer.onload instead since it will give you a callback when ALL buffers are loaded.

code
var buffer = new Tone.Buffer("path/to/sound.mp3", function(){
	//the buffer is now available.
	var buff = buffer.get();
});
default values
url:undefined
onload:Tone.noOp
reverse:false
