JCReverb → Tone.Effect

Tone.JCReverb is a simple Schroeder Reverberator tuned by John Chowning in 1970. It is made up of three allpass filters and four Tone.FeedbackCombFilter.

new Tone.JCReverb ([roomSize])
roomSize {NormalRange|Object}

Coorelates to the decay time.

code
var reverb = new Tone.JCReverb(0.4).connect(Tone.Master);
var delay = new Tone.FeedbackDelay(0.5); 
//connecting the synth to reverb through delay
var synth = new Tone.DuoSynth().chain(delay, reverb);
synth.triggerAttackRelease("A4","8n");
default values
roomSize:0.5
members
.roomSize ~ {NormalRange}

room size control values between [0,1]

.wet ~ {NormalRange}

The wet control is how much of the effected will pass through to the output. 1 = 100% effected signal, 0 = 100% dry signal.
