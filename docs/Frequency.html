Frequency → Tone.TimeBase

Tone.Frequency is a primitive type for encoding Frequency values. Eventually all time values are evaluated to hertz using the eval method.

new Tone.Frequency (val[, units])
val {String|Number}

The time value.

units {String}

The units of the value.

code
Tone.Frequency("C3").eval() // 261
Tone.Frequency(38, "midi").eval() //
Tone.Frequency("C3").transpose(4).eval();
