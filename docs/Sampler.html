Sampler → Tone.Instrument

Sampler wraps Tone.Player in an AmplitudeEnvelope.

new Tone.Sampler (url[, onload])
url {String}

the url of the audio file

onload {function}

The callback to invoke when the sample is loaded.

code
var sampler = new Sampler("./audio/casio/A1.mp3", function(){
	//repitch the sample down a half step
	sampler.triggerAttack(-1);
}).toMaster();
default values
onload:Tone.noOp
loop:false
reverse:false
envelope:{
attack:0.001
decay:0
sustain:1
release:0.1
}
members
.player {Tone.Player}

The sample player.

.envelope {Tone.AmplitudeEnvelope}

The amplitude envelope.

.loop {number|string}

If the output sample should loop or not.

.reverse {boolean}

The direction the buffer should play in

.volume ~ {Decibels}

The volume of the output in decibels.

.triggerAttackRelease (note, duration[, time][, velocity])
note {Frequency}
The note to trigger.

How long the note should be held for before triggering the release.
time = now {Time}
