Emitter → Tone

Tone.Emitter gives classes which extend it the ability to listen for and trigger events. Inspiration and reference from Jerome Etienne's MicroEvent. MIT (c) 2011 Jerome Etienne.

.off (event[, callback])
event {String}
The event to stop listening to.
callback {function}
The callback which was bound to the event with Tone.Emitter.on. If no callback is given, all callbacks events are removed.
