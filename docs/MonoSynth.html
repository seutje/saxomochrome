MonoSynth → Tone.Monophonic

Tone.MonoSynth is composed of one oscillator, one filter, and two envelopes. The amplitude of the Tone.Oscillator and the cutoff frequency of the Tone.Filter are controlled by Tone.Envelopes.

new Tone.MonoSynth ([options])
options {Object}

the options available for the synth see defaults below

code
var synth = new Tone.MonoSynth({
	"oscillator" : {
		"type" : "square"
 },
 "envelope" : {
 	"attack" : 0.1
 }
}).toMaster();
synth.triggerAttackRelease("C4", "8n");
default values
frequency:"C4"
detune:0
oscillator:{
type:"square"
}
filter:{
Q:6
type:"lowpass"
rolloff:-24
}
envelope:{
attack:0.005
decay:0.1
sustain:0.9
release:1
}
filterEnvelope:{
attack:0.06
decay:0.2
sustain:0.5
release:2
baseFrequency:200
octaves:7
exponent:2
}
