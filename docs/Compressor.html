Compressor → Tone

Tone.Compressor is a thin wrapper around the Web Audio DynamicsCompressorNode. Compression reduces the volume of loud sounds or amplifies quiet sounds by narrowing or "compressing" an audio signal's dynamic range. Read more on Wikipedia.

new Tone.Compressor ([threshold][, ratio])
threshold {Decibels|Object}

The value above which the compression starts to be applied.

ratio {Positive}

The gain reduction ratio.

code
var comp = new Tone.Compressor(-30, 3);
