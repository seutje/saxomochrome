Analyser → Tone

Wrapper around the native Web Audio's AnalyserNode. Extracts FFT or Waveform data from the incoming signal.

new Tone.Analyser ([type][, size])
type {String}

The return type of the analysis, either "fft", or "waveform".

size {Number}

The size of the FFT. Value must be a power of two in the range 32 to 32768.

default values
size:1024
returnType:"byte"
type:"fft"
smoothing:0.8
maxDecibels:-30
minDecibels:-100
members
.size {Number}

The size of analysis. This must be a power of two in the range 32 to 32768.

.type {String}

The analysis function returned by Tone.Analyser.analyse(), either "fft" or "waveform".

.smoothing {NormalRange}

0 represents no time averaging with the last analysis frame.

.minDecibels {Decibels}

The smallest decibel value which is analysed by the FFT.

.maxDecibels {Decibels}

The largest decibel value which is analysed by the FFT.
