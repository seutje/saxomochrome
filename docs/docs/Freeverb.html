Freeverb → Tone.Effect

Tone.Freeverb is a reverb based on Freeverb. Read more on reverb on SoundOnSound.

new Tone.Freeverb ([roomSize][, dampening])
roomSize {NormalRange|Object}

Correlated to the decay time.

dampening {Frequency}

The cutoff frequency of a lowpass filter as part of the reverb.

code
var freeverb = new Tone.Freeverb().toMaster();
freeverb.dampening.value = 1000;
//routing synth through the reverb
var synth = new Tone.AMSynth().connect(freeverb);
default values
roomSize:0.7
dampening:3000
members
.roomSize ~ {NormalRange}

The roomSize value between. A larger roomSize will result in a longer decay.

.dampening ~ {Frequency}

The amount of dampening of the reverberant signal.

.wet ~ {NormalRange}

The wet control is how much of the effected will pass through to the output. 1 = 100% effected signal, 0 = 100% dry signal.
