AutoPanner → Tone.Effect

Tone.AutoPanner is a Tone.Panner with an LFO connected to the pan amount. More on using autopanners here.

new Tone.AutoPanner ([frequency])
frequency {Frequency|Object}

Rate of left-right oscillation.

code
//create an autopanner and start it's LFO
var autoPanner = new Tone.AutoPanner("4n").toMaster().start();
//route an oscillator through the panner and start it
var oscillator = new Tone.Oscillator().connect(autoPanner).start();
default values
frequency:1
type:"sine"
depth:1
members
.depth ~ {NormalRange}

The amount of panning between left and right. 0 = always center. 1 = full range between left and right.

.frequency ~ {Frequency}

How fast the panner modulates between left and right.

.wet ~ {NormalRange}

The wet control is how much of the effected will pass through to the output. 1 = 100% effected signal, 0 = 100% dry signal.
