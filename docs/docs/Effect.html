Effect → Tone

Tone.Effect is the base class for effects. Connect the effect between the effectSend and effectReturn GainNodes, then control the amount of effect which goes to the output using the wet control.

new Tone.Effect ([wet])
wet {NormalRange|Object}

The starting wet value.

default values
wet:1
members
.wet ~ {NormalRange}

The wet control is how much of the effected will pass through to the output. 1 = 100% effected signal, 0 = 100% dry signal.
