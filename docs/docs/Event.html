Event → Tone

Tone.Event abstracts away Tone.Transport.schedule and provides a schedulable callback for a single or repeatable events along the timeline.

new Tone.Event (callback, value)
callback {function}

The callback to invoke at the time.

value {*}

The value or values which should be passed to the callback function on invocation.

code
var chord = new Tone.Event(function(time, chord){
	//the chord as well as the exact time of the event
	//are passed in as arguments to the callback function
}, ["D4", "E4", "F4"]);
//start the chord at the beginning of the transport timeline
chord.start();
//loop it every measure for 8 measures
chord.loop = 8;
chord.loopEnd = "1m";
default values
callback:Tone.noOp
loop:false
loopEnd:"1m"
loopStart:0
playbackRate:1
value:null
probability:1
mute:false
humanize:false
members
.callback {function}

The callback to invoke.

.probability {NormalRange}

The probability that the callback will be invoked at the scheduled time.

code
//the callback will be invoked 50% of the time
event.probability = 0.5;
.humanize {Boolean|Time}

If set to true, will apply small (+/-0.02 seconds) random variation to the callback time. If the value is given as a time, it will randomize by that amount.

code
event.humanize = true;
.mute {Boolean}

If mute is true, the callback won't be invoked.

.state {String} READ ONLY

Returns the playback state of the note, either "started" or "stopped".

.loop {Boolean|Positive}

If the note should loop or not between Tone.Event.loopStart and Tone.Event.loopEnd. An integer value corresponds to the number of loops the Event does after it starts.

.playbackRate {Positive}

The playback rate of the note. Defaults to 1.

code
note.loop = true;
//repeat the note twice as fast
note.playbackRate = 2;
.loopEnd {TransportTime}

The loopEnd point is the time the event will loop if Tone.Event.loop is true.

.loopStart {TransportTime}

The time when the loop should start.

.progress {NormalRange} READ ONLY

The current progress of the loop interval. Returns 0 if the event is not started yet or it is not set to loop.
