Panner → Tone

Tone.Panner is an equal power Left/Right Panner and does not support 3D. Panner uses the StereoPannerNode when available.

new Tone.Panner ([initialPan])
initialPan = 0 {NormalRange}

The initail panner value (defaults to 0 = center)

code
//pan the input signal hard right. 
 var panner = new Tone.Panner(1);
members
.pan ~ {AudioRange}

The pan control. -1 = hard left, 1 = hard right.
