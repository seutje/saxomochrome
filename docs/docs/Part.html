Part → Tone.Event

Tone.Part is a collection Tone.Events which can be started/stoped and looped as a single unit.

new Tone.Part (callback, events)
callback {function}

The callback to invoke on each event

events {Array}

the array of events

code
var part = new Tone.Part(function(time, note){
	//the notes given as the second element in the array
	//will be passed in as the second argument
	synth.triggerAttackRelease(note, "8n", time);
}, [[0, "C2"], ["0:2", "C3"], ["0:3:2", "G2"]]);
code
//use an array of objects as long as the object has a "time" attribute
var part = new Tone.Part(function(time, value){
	//the value is an object which contains both the note and the velocity
	synth.triggerAttackRelease(value.note, "8n", time, value.velocity);
}, [{"time" : 0, "note" : "C3", "velocity": 0.9}, 
	   {"time" : "0:2", "note" : "C4", "velocity": 0.5}
]).start(0);
default values
callback:Tone.noOp
loop:false
loopEnd:"1m"
loopStart:0
playbackRate:1
probability:1
humanize:false
mute:false
members
.callback {function}

The callback to invoke at all the scheduled events.

.mute {Boolean}

If mute is true, the callback won't be invoked.

.probability {NormalRange}

The probability of the notes being triggered.

.loop {Boolean|Positive}

If the part should loop or not between Tone.Part.loopStart and Tone.Part.loopEnd. An integer value corresponds to the number of loops the Part does after it starts.

code
//loop the part 8 times
part.loop = 8;
.loopEnd {TransportTime}

The loopEnd point determines when it will loop if Tone.Part.loop is true.

.loopStart {TransportTime}

The loopStart point determines when it will loop if Tone.Part.loop is true.

.playbackRate {Positive}

The playback rate of the part

.length {Positive} READ ONLY

The number of scheduled notes in the part.

.humanize {Boolean|Time}

If set to true, will apply small (+/-0.02 seconds) random variation to the callback time. If the value is given as a time, it will randomize by that amount.

code
event.humanize = true;
.state {String} READ ONLY

Returns the playback state of the note, either "started" or "stopped".

.progress {NormalRange} READ ONLY

The current progress of the loop interval. Returns 0 if the event is not started yet or it is not set to loop.

.at (time[, value])
time {TransportTime}
The time of the event to get or set.
value {*}
If a value is passed in, the value of the event at the given time will be set to it.
returns {Tone.Event}

the event at the time

Get/Set an Event's value at the given time. If a value is passed in and no event exists at the given time, one will be created with that value. If two events are at the same time, the first one will be returned.

code
part.at("1m"); //returns the part at the first measure

part.at("2m", "C2"); //set the value at "2m" to C2. 
//if an event didn't exist at that time, it will be created.
