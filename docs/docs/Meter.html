Meter → Tone

Tone.Meter gets the RMS of an input signal with some averaging applied. It can also get the raw value of the input signal.

new Tone.Meter (type, smoothing)
type {String}

Either "level" or "signal".

smoothing {Number}

The amount of smoothing applied between frames.

code
var meter = new Tone.Meter();
var mic = new Tone.UserMedia().start();
//connect mic to the meter
mic.connect(meter);
//the current level of the mic input
var level = meter.value;
default values
smoothing:0.8
type:Tone.Meter.Type.Level
members
.type {String}

The type of the meter, either "level" or "signal". A "level" meter will return the volume level (rms) of the input signal and a "signal" meter will return the signal value of the input.

.smoothing {Number}

The amount of carryover between the current and last frame. Only applied meter for "level" type.

.value {Number} READ ONLY

The current value of the meter. A value of 1 is "unity".
