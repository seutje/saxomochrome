Volume → Tone

Tone.Volume is a simple volume node, useful for creating a volume fader.

new Tone.Volume ([volume])
volume = 0 {Decibels}

the initial volume

code
var vol = new Tone.Volume(-12);
instrument.chain(vol, Tone.Master);
default values
volume:0
mute:false
members
